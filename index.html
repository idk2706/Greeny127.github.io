Some of my projects

Hair Color Changer

Uses python and opencv. Give it an input image and it'll detect the hair and change it's color. The way I find the hair is through detecting the face and eyes, then use those locations to find the hair. It works on most images, but not well on ones which are blurry/sideways! Still, it's pretty cool considering it doesn't use Machine Learning to detect the hair.

Particle Simulator

Uses C++ and SFML. Very fun to play around with and it was a new learning experience for me since it was my first time properly using c++. It has various other particles such as water, wood and fire. Really cool project!

Rom Hacking

Don't worry, it's not hacking! Rom hacking is essentially modding a rom to add your own sprites, maps or features. I learnt assembly through rom hacking and it was really fun! I also was able to understand how games were made back in the GameBoy Era.

Animal Detection

Python and keras + tensorflow. I learnt how to use Keras and decided to build this project with the basic knowledge I gained while learning Deep Learning. I was stumped many times at the accuracy but with trial and error I eventually got it to 97% accuracy! It can detect Dogs, Cats, Rabbits, Lions and Tigers.
